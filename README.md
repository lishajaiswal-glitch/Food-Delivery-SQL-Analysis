# 🍽️ Zomato-Style Food Delivery Data Analysis (SQL Project)

## 📌 Project Overview

This project analyzes a food delivery platform inspired by Zomato using SQL.
The objective is to extract meaningful insights from structured data and understand key business areas such as customer behavior, restaurant performance, and delivery operations.

---

##  Business Problem

Food delivery platforms generate large volumes of data, but without proper analysis, it is difficult to:

* Identify active and inactive customers
* Evaluate restaurant performance
* Understand order trends over time
* Measure delivery partner efficiency

This project addresses these challenges using SQL-based data analysis.

---

##  Sample Advanced Query

```sql
SELECT 
    restaurant_id,
    AVG(rating) AS avg_rating,
    RANK() OVER (ORDER BY AVG(rating) DESC) AS ranking
FROM ratings
GROUP BY restaurant_id;
```

This query ranks restaurants based on their average ratings, helping identify top-performing businesses.
